# RWYEngine

RWYEngine is a desktop tool for writers to store, search, and visualize information about their stories.  
It lets you keep track of characters, locations, items, events, and the relationships between them.

## Features
- Create and edit entries (title, description, category, tags, synonyms)
- Visual relationship map (tree or force layout)
- Drag nodes with live edge updates
- Export relationship map as PNG
- Autosaves when switching entries

## Installation
Download and run the installer from the releases page.  
RWYEngine will create its database automatically on first launch.
